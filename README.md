# clojuretutorial

This project template is generated by calling:
<pre><code> lein new clojuretutorial </code></pre>

After adding dependencies on project.clj, get dependencies:
<pre><code>
lein deps 
</code></pre>

## Usage

To run web application.
<pre><code>
lein ring server 3000
</code></pre>
